The _bot_training dirs each contain a csv_files subdir and a jsonl_files subdir, which contain the relevant csv and python files as well as the converted jsonl files, respectively.

The inference_instructions directory contains the instructions used during deployment to the website.

The training_instructions directory contains the instructions used during training.

The prompts directory contains the prompts, which are like prepended to the chat history.

In each of the dirs, there are files with 'main' in the name. These EXACT files are targeted in the chat_application as well as the csv_to_jsonl converter. Therefore, whatever final version of the prompts and instructions we want must be in those "main" files.
